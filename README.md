# Affirmations Gumball

A soft pastel pixel-art affirmation gumball machine made with plain HTML, CSS, and JavaScript.

## Features

- Pixel-style pastel UI
- Custom gumball machine artwork
- Click the red button to dispense a gumball
- Soft pop and sparkle sounds
- Slow affirmation fade-in with glow
- Lightweight single-file frontend

## Files

- `index.html` - the full app
- `machine.png` - the gumball machine artwork

## How to Use

1. Download or clone the project.
2. Open `index.html` in a browser.
3. Click the red button on the machine.

## Customization

You can edit the affirmations directly inside `index.html` in the `affirmations` array.
